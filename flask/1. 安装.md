# 1. 安装

### 安装virtualenv：
> pip install virtualenv

1.命名虚拟环境：
> virtualenv demo

此文件夹下出现demo文件夹

2.激活：
> demo\Scripts\activate

前面会出现（demo）


### 安装flask：
> pip install flask

如果没有报错信息，我们的安装就成功了。

### 测试：

新建hello.py
```
from flask import Flask
app = Flask(__name__)

@app.route('/')
def index():
	return '<h1>hello,python!</h1>'

if __name__ == '__main__':
	app.run(debug=True)
```

> python hello.py

会看到
> running on http://127.0.0.1:5000

浏览器打开这个位置：
你会看到hello,python，说明我们环境安装正常。
